# Protocol 

### Before starting 
1. Make sure you digest 
2. ligate using the correct pmol 
3. 2:1 or 3:1 ratio fragment:backbone


| Component | Amount | 
| ---- | ---- | 
| T4/T7 Ligase Buffer | 2 uL  | 
| DNA 1  | 1x pmol | 
| DNA 2  | 2 or 3x pmol | 
| H2O | to 20 uL  | 
| T4/T7 DNA ligase | 1 uL | 

Total: 20 uL 

- Add T4/T7 ligase last 

### Incubation:  

16C for 24 h 

65C for 10 min 

10C forever 

Cleanup and Elute
